# demo-for-shasha

## 开发环境

1）安装node v6.3.1

2）安装webpack和webpack-dev-server
npm install  

3）启动服务器
npm start

4）在浏览器输入
http://localhost:1314

5）继续开发

## github 操作

1）fork项目

2）clone 到本地
git clone

3）切换分支
git checkout -b origin/gh-pages

4）开发代码

5）提交代码
git add --all
git commit -m "xxxxx"
git push -u origin/gh-pages
